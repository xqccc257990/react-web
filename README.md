# react-r3-saga
可能是最简单React+Redux+Router+Redux-saga+AntD+ES6全家桶

## 运行项目（nodejs 6.0+）

```
 git clone https://github.com/wlc534/react-r3-saga

 cd react-r3-saga

 npm install
  
 npm start

 
```
